Trabalhando com operador spread

Qual a função do opreador spread?

Ele é usado por três pontinhos atecedendo minha array, note no exemplo que dentro da array peças coloquei mais duas outras array com os tres pontinhos antes, esses três pontinhos vai organizar todos os dados da minha array peças, alinhando como q fosse uma única array, isso é muito útil quando se quer atualizar algo incluindo uma nova array
